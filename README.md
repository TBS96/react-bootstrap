# React + Vite + Bootstrap

## Install Bootstrap
```bash
npm i react-bootstrap bootstrap
```
## Importing the Bootstrap CSS in `main.jsx`
```bash
import 'bootstrap/dist/css/bootstrap.min.css'
```
